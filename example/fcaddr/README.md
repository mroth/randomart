# fcaddr

> Example tool to generate a vizhash from a ⨎ Filecoin $FIL f1 address.

## Usage
```
Usage of fcaddr:
  -tiles string
        openssh|galaxy (default "galaxy")
  -x uint
        width (cols) (default 10)
  -y uint
        height (rows) (default 10)
```
